# 🚀 Workflow Thief Arena - 快速使用指南

## 快速开始

### 方法 1: Web UI (推荐)

1. **双击运行** `启动WebUI.bat`（Windows）
   或在命令行运行:
   ```bash
   python app.py
   ```

2. **打开浏览器访问**: http://localhost:5000

3. **使用界面**:
   - 点击预设按钮查看示例
   - 或输入关键词搜索任意工作流
   - 开关「使用真实 LLM 分析」切换模式
   - 点击「开始分析」查看结果

### 方法 2: 命令行

**预设数据模式**（无需API）:
```bash
python agent_enhanced.py --preset "招聘筛选流程"
```

**真实 LLM 模式**（需要配置API）:
```bash
python agent_enhanced.py "招聘筛选流程"
```

## API 配置

如需使用真实 LLM 分析，请编辑 `.env` 文件:

```env
VOLCENGINE_API_KEY=你的_api_key
BASE_URL=https://ark.cn-beijing.volces.com/api/v3
MODEL_NAME=你的_model_id
```

详细配置说明请查看 `API_SETUP_GUIDE.md`

## 常见问题

**Q: 没有 API Key 可以使用吗？**

A: 可以！关闭「使用真实 LLM 分析」开关，使用预设数据模式即可。

**Q: API 连接失败怎么办？**

A: 系统会自动回退到预设数据模式，不影响使用。

**Q: 如何查看生成的报告？**

A: Web UI 分析完成后可以直接下载，或在 output/ 目录查看。

**Q: 支持自定义工作流分析吗？**

A: 支持！在输入框输入任意关键词即可搜索分析。
