# 截图目录

本目录存放产品页面截图，供 `docs/product-pages.md` 引用。

## 命名规则

`序号-页面名.png`，例如 `01-login.png`、`05-detail-timeline.png`。

## 截图清单

| 文件 | 页面 | 说明 |
|------|------|------|
| 01-login.png | 登录页 | 邮箱密码表单 |
| 02-register.png | 注册页 | 首用户自动 admin 提示 |
| 03-dashboard.png | Dashboard | 工作流列表 + 用量卡片 |
| 04-new-workflow.png | 新建工作流 | 表单 + 示例按钮 |
| 05-detail-timeline.png | 工作流详情 | 时间线 Tab |
| 06-timeline-expanded.png | 时间线展开 | 展开某条 tool_call |
| 07-evidence.png | 证据链 | 证据列表 + score 徽章 |
| 08-report.png | 报告 | final_output JSON |
| 09-usage.png | 用量统计 | 今日 + 本月卡片 |
| 10-navbar.png | 顶栏 | 导航 + 用户菜单 |

## 截图方法

1. 启动完整应用
2. 浏览器打开 http://localhost:5173
3. 按清单顺序访问页面
4. Chrome DevTools → More tools → Capture screenshot（或 Win+Shift+S）
5. 保存到此目录，按命名规则

建议分辨率 1440×900。
